# Apache Wicket (apache-wicket)
Apache Wicket is a component-based web application framework for Java that provides a clean separation of markup and logic with a stateful component model. It enables developers to build web applications using pure Java and HTML.

### Apache Wicket
Wicket provides a Java component-based web framework API with a stateful programming model, pure HTML templates, built-in Ajax support, and an extensible component hierarchy for building type-safe web applications.
